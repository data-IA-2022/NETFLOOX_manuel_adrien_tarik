Foire aux questions

  • Netfloox est un système de recommandation de contenu vidéo.
    Le projet s'articule autour de plusieurs axes:

    1. Analyse des données récupérées de fichiers tsv issue de la base IMDB.

    2. Processing des données.

    3. Création d'un modèle de base de données relationnel.

    4. Création de deux modèles de machine learning:
         1. Un modèle de Prédiction de popularité
         2. Un modèle pour recommender des films simmilaires.

    5. Une application web pour que les utilisateurs interagissent avec la base et visualisent les recommendations en fonction de leurs inputs.
